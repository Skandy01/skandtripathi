### 📌 Recent Enterprise Project – Hybrid Cloud 5G Network Monitoring Platform (Germany Telecom) 

## Currently leading the DevOps transformation for a major German telecom operator.  
Migrating their legacy 5G network monitoring system from fully on-premises (Ericsson hardware) to a secure **hybrid cloud architecture** using Azure Kubernetes Service (AKS).

**Platform Overview**  
- React-based real-time dashboards running as containerised microservices on AKS  
- Python backend (data ingestion, processing, API layers) fully containerised  
- Azure SQL with private endpoints + legacy VMs connected via ExpressRoute  

**My Key Contributions**  
- Designed & built complete Azure platform foundation using modular Terraform (hub-and-spoke, AKS, Application Gateway, private endpoints)  
- Wrote optimised multi-stage Dockerfiles + integrated security scanning pipelines  
- Engineered enterprise DevSecOps CI/CD in Azure DevOps + GitHub Actions (SAST/DAST, Helm deployments, blue-green strategy)  
- Implemented full-stack observability across VMs and pods (Azure Monitor + Prometheus/Grafana)  
- Delivered **50% faster incident response**, zero-downtime releases and seamless scaling during peak 5G traffic  

**Tech Stack**: Terraform • AKS • Kubernetes • Helm • Docker • Azure DevOps • GitHub Actions • Key Vault • Prometheus • Grafana • SonarQube
